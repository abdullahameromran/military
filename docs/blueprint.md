# **App Name**: DutyNotifier

## Core Features:

- Admin Dashboard: Secure admin interface to manage availability.
- Add Free Days: Calendar-based interface to select and save free days.
- Public Status Page: Display current availability status to the public.
- Real-time Status Check: Determine availability based on current date vs. stored free days using Supabase.
- Supabase Integration: Utilize Supabase to store and retrieve availability data, ensure row level security, and manage users.
- AI Availability Suggestion: Tool to suggest potential free days based on past availability patterns and common military leave schedules.

## Style Guidelines:

- Primary color: Dark blue (#2E3148), providing a professional and reliable feel.
- Background color: Light gray (#E8E8EA), for a clean and neutral backdrop.
- Accent color: Teal (#46B29D), used for highlighting key interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif, will be used for both headlines and body text to create a modern, machined, objective look.
- Use simple, clear icons for availability status (e.g., calendar, checkmark).
- Clean, mobile-friendly design with a focus on readability and ease of use.
- Subtle transitions for status updates and calendar interactions.
